# Context7 First Rule

Antes de implementar qualquer funcionalidade:

1. Consultar a documentação mais recente usando Context7.
2. Analisar a arquitetura atual do projeto.
3. Verificar componentes existentes.
4. Verificar hooks existentes.
5. Verificar serviços existentes.
6. Verificar padrões existentes.
7. Criar plano de implementação.
8. Apenas depois implementar.

Nunca assumir APIs ou bibliotecas sem consultar Context7.

Para React, Vite, TypeScript, Tailwind, Shadcn, Motion, Lucide, TanStack Query, React Router e Axios, Context7 é a fonte principal de documentação.
